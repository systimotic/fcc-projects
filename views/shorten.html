URL Shortener API project

Challenge link: URL Shortener Microservice

Usage

You can create a new shortened URL, or use an existing one.

Generating a new shortened URL

To create a new shortened URL, make a GET request to https://systimotic-fcc.herokuapp.com/shorten/new/[yourURL]

Example URL: https://systimotic-fcc.herokuapp.com/shorten/new/https://freecodecamp.com

Output:

    
      {
        "original_url": "https://freecodecamp.com",
        "short_url": "http://systimotic-fcc.herokuapp.com/shorten/L5pUv5k9pV"
      }

Using a shortened URL

Using a shortened URL will redirect you to the original URL.

Example: http://systimotic-fcc.herokuapp.com/shorten/L5pUv5k9pV redirects to https://freecodecamp.com

Errors

Invalid URL

If you pass an invalid URL, you get an error.

Example URL: https://systimotic-fcc.herokuapp.com/shorten/new/freecodecamp

    
      {
        "error": "Wrong URL format. Please enter a valid URL and try again."
      }

Unknown ID

If you browse to a URL with an unknow ID, you get an error

Example URL: https://systimotic-fcc.herokuapp.com/shorten/not-a-valid-ID

    
      {
        "error": "There is no link with this ID."
      }
