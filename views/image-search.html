Image search API project

Challenge link: Image Search Abstraction Layer

Usage

You can view recent searches, or search for an image yourself.

Viewing recent searches

To view recent searches, make a GET request to https://systimotic-fcc.herokuapp.com/image-search/recent

You will get a JSON response, containing an array of objects with the query and the date of the search. Only the 10 most recent searches are displayed.

Output:

    
    [
      {
        "query": "freecodecamp",
        "date": "2016-10-25T18:01:01.662Z"
      },
      {
        "query": "tomatoes",
        "date": "2016-10-25T18:00:48.832Z"
      },
      (...)
    ]

Searching for images

To do a new searches, make a GET request to "https://systimotic-fcc.herokuapp.com/image-search/[your search term]

You will get JSON response, containing an array of objects with a link to the image, a title, and the page the image was found on. Only the 10 most relevant searches are displayed.

Example URL: https://systimotic-fcc.herokuapp.com/image-search/freecodecamp

Output:

    
    [
      {
          "image": "https://www.bing.com/cr?IG=B46FCA8262474BC8(...)",
          "alt": "Freecodecamp",
          "source": "https://www.bing.com/cr?IG=B46FCA8262474BC8(...)"
      },
      {
          "image": "http://www.bing.com/cr?IG=B46FCA8262474BC8(...)",
          "alt": "freeCodeCamp - Bay Area Codes",
          "source": "http://www.bing.com/cr?IG=B46FCA8262474BC8(...)"
      },
      (...)
    ]

Going to another page

Your search will display only the 10 most relevant results. To view the next results, add the ?offset=[page] parameter. The first page is page 0. The maximum is set at 100. If no more results are found, an empty array is returned.

Example URL: https://systimotic-fcc.herokuapp.com/image-search/freecodecamp?offset=1
