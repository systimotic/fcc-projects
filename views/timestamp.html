Timestamp API project

Challenge link: Timestamp Microservice

Usage

You can pass a parameter that is either a Unix timestamp or a natural language date.

Unix timestamp

Here's what a Unix timestamp looks like: 2476915200

Example URL: https://systimotic-fcc.herokuapp.com/timestamp/2476915200

Output:

    
      {
        "unix": 2476915200,
        "natural": "June 28, 2048"
      }

Natural language date

Here's what a natural language date looks like: August 5, 2005

Example URL: https://systimotic-fcc.herokuapp.com/timestamp/August 5, 2005

Output:

    
      {
        "unix": 1123200000,
        "natural": "August 5, 2005"
      }

Errors

Invalid dates

If you pass an invalid date, the values in the object are set to null.

    
      {
        "unix": null,
        "natural": null
      }
